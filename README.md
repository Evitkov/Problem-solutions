# Problem-Solutions
An archive of Codeforces, CSES, and LeetCode problems that taught me something theoretically new.

## Why does this repo exist
This is my personal repository with solutions to problems I found theoretically new to me. I wanted to improve my memorization by describing and saving the ideal solution, and, if applicable, relating that to my original solution and why it failed.

This repository does not contain every problem I solve. If a file is in here, it means it forced me to learn a new algorithmic concept, mathematical trick, or data structure. It acts as my personal knowledge base so I don't forget the logic.

## Repository Structure
The folders are organized by **Algorithm/Topic**, and the names of the files contain their source (e.g., Codeforces, CSES). 

## File Format (Words > Code)
Every `.cpp` file in this repository starts with a comment block explaining the core logic. The code below it includes inline comments if a specific implementation detail was tricky to write, or is left clean and uncommented if the challenge was purely conceptual.
